# 16S amplicon dataset from yellow monkeyflower (Mimulus guttatus)

A phyloseq containing 11085 ASVs (Amplicon Sequence Variants) across 39
samples.

## Usage

``` r
data("mimulus")
```

## Format

An object class "phyloseq".

## Source

Internal test dataset.

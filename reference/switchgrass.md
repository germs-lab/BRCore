# 16S amplicon dataset from Switchgrass (Panicum virgatum)

A phyloseq containing 706 97% sequence similarity OTUs (Operational
Taxonomic Units) across 43 samples.

## Usage

``` r
data("switchgrass")
```

## Format

An object class "phyloseq".

## Source

Internal test dataset.

# 16S amplicon dataset from common bean (Phaseolus vulgaris)

A phyloseq containing 24885 ASVs (Amplicon Sequence Variants) across 30
samples.

## Usage

``` r
data("bean")
```

## Format

An object class "phyloseq".

## Source

Internal test dataset.
